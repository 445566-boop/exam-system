## 2. 题库上传流程

```
用户                    前端                    后端                    LLM                    数据库
 │                      │                       │                       │                       │
 │  上传Word文档        │                       │                       │                       │
 │────────────────────>│                       │                       │                       │
 │                      │  发送文件到服务器      │                       │                       │
 │                      │─────────────────────>│                       │                       │
 │                      │                       │  mammoth解析文档      │                       │
 │                      │                       │──────────────────────>│                       │
 │                      │                       │<──────────────────────│                       │
 │                      │                       │    返回纯文本内容      │                       │
 │                      │                       │                       │                       │
 │                      │                       │  调用LLM解析题目      │                       │
 │                      │                       │─────────────────────>│                       │
 │                      │                       │<─────────────────────│                       │
 │                      │                       │  返回结构化题目JSON   │                       │
 │                      │                       │                       │                       │
 │                      │                       │  插入数据库            │                       │
 │                      │                       │────────────────────────────────────────────>│
 │                      │                       │                       │                       │
 │  返回解析结果         │                       │                       │                       │
 │<─────────────────────│                       │                       │                       │
 │                      │                       │                       │                       │
```

```
 │                      │                       │                       │

 │────────────────────>│                       │                       │
 │                      │  发送配置到服务器       │                       │
 │                      │─────────────────────>│                       │
 │                      │                       │                       │
 │                      │                       │  查询符合条件的题目    │
 │                      │                       │────────────────────────────────────────────>│
 │                      │                       │                       │
 │                      │                       │<────────────────────────────────────────────│
 │                      │                       │    返回题目列表       │
 │                      │                       │                       │
 │                      │                       │  随机抽取指定数量     │
 │                      │                       │──────────────────────│
 │                      │                       │                       │
 │                      │                       │  docx生成Word文档     │
 │                      │                       │──────────────────────│
 │                      │                       │                       │
 │                      │                       │  上传到存储           │
 │                      │                       │────────────────────────────────────────────>│
 │                      │                       │                       │
 │                      │                       │<────────────────────────────────────────────│
 │                      │                       │    返回下载链接       │

```

## 4. 试卷批改流程

```
用户                    前端                    后端                    LLM                    数据库
 │                      │                       │                       │                       │
 │  上传已作答试卷        │                       │                       │                       │
 │────────────────────>│                       │                       │                       │
 │                      │  发送答题卡到服务器    │                       │                       │
 │                      │─────────────────────>│                       │                       │
 │                      │                       │  mammoth解析答题卡    │                       │
 │                      │                       │──────────────────────>│                       │
 │                      │                       │<──────────────────────│                       │
 │                      │                       │    返回用户答案       │
 │                      │                       │                       │
 │                      │                       │  获取题库答案        │
 │                      │                       │────────────────────────────────────────────>│
 │                      │                       │                       │
 │                      │                       │<────────────────────────────────────────────│
 │                      │                       │    返回标准答案       │
 │                      │                       │                       │
 │                      │                       │  调用LLM批改         │
 │                      │                       │─────────────────────>│                       │
 │                      │                       │<─────────────────────│                       │
 │                      │                       │  返回批改结果+错题   │
 │                      │                       │                       │
 │                      │                       │  保存错题到数据库    │
 │                      │                       │────────────────────────────────────────────>│
 │                      │                       │                       │
 │  返回批改结果         │                       │                       │
 │<─────────────────────│                       │                       │
 │                      │                       │                       │
```

## 5. 数据流向总览

```
                           ┌─────────────────┐
                           │    用户上传题库    │
                           │   (Word文档)      │
                           └────────┬────────┘
                                    │
                                    ▼
                    ┌───────────────────────────────┐
                    │     1. mammoth 解析           │
                    │        Word → 纯文本           │
                    └───────────────┬───────────────┘
                                    │
                                    ▼
                    ┌───────────────────────────────┐
                    │     2. LLM 智能解析            │
                    │   纯文本 → 结构化题目 JSON      │
                    └───────────────┬───────────────┘
                                    │
                    ┌───────────────┴───────────────┐
                    │                               │
                    ▼                               ▼
           ┌─────────────────┐             ┌─────────────────┐
           │   question_bank  │             │    统计聚合     │
           │     (题库表)     │             │  (题型/难度)    │
           └────────┬────────┘             └────────┬────────┘
                    │                               │
                    └───────────────┬───────────────┘
                                    │
                                    ▼
                    ┌───────────────────────────────┐
                    │         数据展示层               │
                    │   柱状图 + 饼图 + 统计卡片       │
                    └───────────────────────────────┘

                           ┌─────────────────┐
                           │    组卷/批改请求  │
                           └────────┬────────┘
                                    │
                    ┌───────────────┴───────────────┐
                    │                               │
                    ▼                               ▼
           ┌─────────────────┐             ┌─────────────────┐
           │     组卷流程     │             │     批改流程     │
           │  数据库查询      │             │ LLM智能批改     │
           │  随机抽取       │             │ 错题记录       │
           │  生成Word      │             │ 结果展示       │
           └─────────────────┘             └─────────────────┘
```
